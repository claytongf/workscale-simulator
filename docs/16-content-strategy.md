# 16 — Estratégia de conteúdo

## Objetivo

Usar o projeto para gerar engajamento em redes sociais, vídeos e posts, conectando tecnologia, games, produtividade e debate sobre trabalho.

## Ideia central

> Transformei escalas de trabalho em um simulador estilo videogame.

## Títulos possíveis

```txt
Transformei a escala 6x1 em um videogame
Simulei 1 ano de trabalho 6x1 vs 5x2
E se sua energia no trabalho fosse uma barra de Street Fighter?
Quanto uma empresa gastaria mudando de 6x1 para 5x2?
Trabalhar menos pode produzir mais? Simulei no Python
O preço dos produtos subiria com o fim da escala 6x1?
Simulei o impacto das férias na energia de um trabalhador
Brasil vs EUA: comparei jornadas de trabalho em um simulador
```

## Estrutura de vídeo

```txt
1. Gancho
2. Explicação da barra de energia
3. Configuração do cenário
4. Simulação rodando
5. Gráficos aparecendo
6. Comparação 6x1 vs 5x2
7. Custo da empresa
8. Preço final
9. Limitações
10. Chamada para debate
```

## Chamada para GitHub

```txt
O código está aberto no GitHub para quem quiser estudar, criticar, melhorar os modelos ou sugerir novos cenários.
```

## Possíveis posts

```txt
print da barra de energia depois de 365 dias
comparação entre 6x1 e 5x2
gráfico de burnout acumulado
impacto das férias
quanto o deslocamento muda o salário/hora real
quanto o preço final muda se a empresa contratar mais pessoas
```

## Tom de comunicação

- educativo;
- curioso;
- visual;
- sem militância direta;
- aberto a críticas;
- transparente sobre limitações;
- com humor leve quando fizer sentido.

## Compartilhamento de Relatórios Estruturados

Para facilitar que criadores de conteúdo, pesquisadores e usuários em redes sociais compartilhem os dados de simulações em posts de blog ou discussões, a Fase 9 permite baixar relatórios estruturados em Markdown e CSV. O arquivo Markdown gerado pode ser colado diretamente no Reddit, GitHub, blogs ou fóruns de debate, sempre carregando de forma transparente a nota de limitações éticas e científicas.
