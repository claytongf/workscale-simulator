# 01 — Visão geral

O **WorkScale Simulator** é uma aplicação open source criada para simular e comparar diferentes escalas de trabalho ao longo do tempo.

O objetivo é visualizar como jornadas como **6x1, 5x2, 4x3, 12x36, freelancer, part-time ou hourly work** podem impactar:

- energia do trabalhador;
- recuperação física e mental;
- produtividade;
- produção bruta e líquida;
- risco de burnout;
- tempo livre;
- renda mensal;
- custo para a empresa;
- encargos/impostos trabalhistas;
- custo por unidade produzida;
- preço final de produtos ou serviços;
- comparação entre países.

## Proposta central

A aplicação simula o trabalhador como um personagem com atributos variáveis.

O trabalhador possui uma barra de energia, semelhante a jogos de luta ou esportes. Essa barra diminui durante trabalho, deslocamento e tarefas obrigatórias, e aumenta durante sono, descanso e férias.

Ao mesmo tempo, a empresa possui custos e produção. Mudanças na escala de trabalho podem afetar:

- horas trabalhadas;
- produção total;
- necessidade de contratar mais pessoas;
- custo por funcionário;
- custo por unidade;
- preço final.

## O que o projeto não é

O projeto não deve ser tratado como:

- ferramenta jurídica;
- ferramenta médica;
- ferramenta econômica definitiva;
- calculadora oficial de encargos;
- prova de que uma escala é universalmente melhor que outra.

O projeto é uma simulação educacional baseada em parâmetros ajustáveis.
