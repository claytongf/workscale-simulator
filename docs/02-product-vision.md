# 02 — Visão de produto

## Nome do projeto

**WorkScale Simulator**

## Descrição curta

Simulador educacional para comparar escalas de trabalho, energia humana, produtividade, custo empresarial e impacto no preço final.

## Público-alvo

O projeto pode ser útil para:

- trabalhadores;
- estudantes;
- criadores de conteúdo;
- profissionais de RH;
- pequenos empresários;
- economistas;
- desenvolvedores;
- professores;
- pessoas interessadas em produtividade e qualidade de vida.

## Posicionamento

O projeto deve ser apresentado como um simulador de cenários, não como uma ferramenta ideológica.

Mensagem central:

> O WorkScale Simulator permite comparar cenários de trabalho com base em energia, produtividade, custos e preço final, usando parâmetros ajustáveis e modelos transparentes.

## Diferenciais

- Visual gamificado com barra de energia.
- Comparação entre escalas.
- Simulação ao longo do tempo.
- Inclusão de férias.
- Inclusão de produtividade individual.
- Inclusão de experiência.
- Inclusão de custos empresariais.
- Inclusão de encargos/impostos trabalhistas.
- Comparação entre países.
- Código aberto para estudo, crítica e contribuição.

## Princípios de produto

1. Transparência acima de conclusões prontas.
2. Parâmetros editáveis.
3. Linguagem neutra.
4. Visual simples e didático.
5. Fórmulas documentadas.
6. Resultados apresentados como estimativas.
