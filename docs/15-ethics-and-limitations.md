# 15 — Ética e limitações

## Aviso principal

Este simulador é educacional e experimental.

Ele não substitui consultoria:

- trabalhista;
- econômica;
- médica;
- psicológica;
- contábil;
- jurídica.

Os resultados dependem das premissas inseridas pelo usuário. Energia, produtividade, burnout, custos e preços são estimativas baseadas em modelos simplificados.

O objetivo do projeto é permitir comparação de cenários, não determinar qual escala de trabalho é universalmente melhor.

## Linguagem neutra

O sistema não deve julgar moralmente trabalhadores.

Evitar termos como:

```txt
preguiçoso
vagal
inútil
fraco
```

Usar termos como:

```txt
produtividade baixa
produtividade média
produtividade alta
produtividade excepcional
```

## Princípios

```txt
Produtividade não representa valor humano.
Experiência não representa inteligência.
Energia não representa força de vontade.
Baixa produção pode ser consequência de fatores pessoais, ambientais, técnicos, econômicos ou organizacionais.
```

## Limitações do modelo

O simulador não consegue representar perfeitamente:

- saúde individual;
- problemas familiares;
- alimentação real;
- contexto econômico;
- cultura organizacional;
- assédio;
- insegurança financeira;
- motivação;
- depressão;
- ansiedade;
- desigualdade social;
- informalidade;
- diferenças legais completas entre países.

## Como apresentar os resultados

Sempre apresentar como:

```txt
estimativa
cenário simulado
resultado dependente das premissas
modelo simplificado
```

Nunca apresentar como:

```txt
verdade absoluta
prova definitiva
recomendação legal
diagnóstico médico
```

## Nota sobre Presets de Países

> [!WARNING]
> Os presets de países são estimativas educacionais editáveis e não substituem assessoria profissional jurídica, contábil, tributária, trabalhista ou econômica.

## Nota sobre Exportação de Relatórios

> [!IMPORTANT]
> Todo relatório exportado (JSON, CSV, Markdown) contém obrigatoriamente a nota de limitação educacional e ética. Os resultados contidos no relatório não representam conclusões científicas, médicas, contábeis ou de recursos humanos.
